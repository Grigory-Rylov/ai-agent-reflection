# О проекте ai-agent-reflection

## Что это такое

Это **AI Agent Gateway (VK Bot)** — шлюз на языке Go, который соединяет VK Bot API с локальными LLM-моделями (llama-server).

## Основная архитектура

```
VK Bot API → Agent Gateway → llama-server (локальная LLM)
                │
         ┌──────┼──────┐
         │      │      │
     Session  Tools   VK Client
```

## Ключевые компоненты

1. **vk-gateway** — основной AI-агент с поддержкой:
   - Стриминг ответов
   - Function calling
   - 11 встроенных инструментов:
     - Файловые операции
     - Выполнение shell-команд
     - Веб-поиск/получение данных
     - Поиск кода
     - Математические вычисления
     - И другие

2. **vk-gateway-restarter** — watchdog для удалённого управления через VK-команды:
   - `/update` — git pull, пересборка, перезапуск
   - `/b <branch>` — переключение на ветку
   - `/restart` — перезапуск без пересборки

## Команды для бота

| Команда | Описание |
|---------|----------|
| `/reset` | Очистить историю диалога |
| `/newsession [path]` | Новый диалог с изменением рабочей директории |
| `/status` | Показать информацию о сессии |
| `/help` | Список команд |

## Стек технологий

- **Go** — основной язык реализации
- **VK Bot API** — интерфейс взаимодействия
- **llama-server** — локальная LLM-модель
- **JSON Config** — конфигурация через `config.json`

## Для чего используется

Проект позволяет создать умного AI-ассистента в VK, который может:
- Выполнять команды на сервере
- Работать с файлами
- Искать информацию в интернете
- Анализировать код
- Делать математические вычисления

Всё это через обычный чат в VK с поддержкой локальных LLM-моделей.
